Setup the node for the validation process.

Usage:
  opct adm setup-node [flags]

Examples:
opct adm setup-node

Flags:
  -h, --help          help for setup-node
      --node string   Node to set required label and taints
  -y, --yes           Node to set required label and taints

Global Flags:
      --kubeconfig string   kubeconfig for target OpenShift cluster
      --log-level string    logging level (default "info")
